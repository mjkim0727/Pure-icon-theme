# Pure-icon-theme
Mac-Like icon theme for Linux Desktops

## Features

* Mac-Like Designs
* KDE 6 Support
* iOS18-Style Dark Mode icons

## Based On

* Fluent-icon-theme by vinceliuice - almost symbolic icons
* Stylish-icon-theme by mjkim0727

## Install

### via sources

Open terminal, and run install.sh file.

```
$ ./install.sh
```
